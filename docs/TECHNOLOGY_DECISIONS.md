# Technology Decisions

## Next.js 15 App Router

Next.js provides production routing, optimized bundling, metadata support, and deployment-friendly defaults. The App Router keeps the main page composition clear and future-ready.

## TypeScript

Strict TypeScript keeps section data, icons, social links, and GitHub responses predictable as the portfolio grows.

## Tailwind CSS

Tailwind supports a disciplined dark design system with fast responsive iteration. The palette is intentionally minimal: black, white, muted gray, and purple accent.

## Framer Motion

Framer Motion powers scroll-triggered fade, reveal, stagger, slide, and scale animations with reusable variants.

## Three.js and React Three Fiber

The global particle field is rendered with React Three Fiber on top of Three.js for high-performance GPU rendering and smooth mouse-reactive motion.

## React Icons

React Icons provides consistent technology, social, and interface icons without hand-drawn SVG maintenance.

## EmailJS

EmailJS enables a static-site-compatible contact form. Credentials are stored in public environment variables as required by EmailJS browser usage.
